# 🛒 TOKOku - Simple E-Commerce Web Application

TOKOku is a front-end e-commerce web application built using HTML, CSS, and JavaScript.  
This project simulates a basic online shopping experience and is currently under active development.

⚠️ This project is still in progress and continuously being improved.

---

## Current Features

- Homepage with product listing
- Product detail page
- Add to Cart functionality
- Add to Wishlist functionality
- Account page navigation
- LocalStorage-based state management
- Responsive layout (basic)

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

---

## Project Goals

This project was built to:

- Practice front-end development fundamentals
- Implement DOM manipulation using JavaScript
- Simulate basic e-commerce flow
- Improve UI/UX design skills
- Build a scalable structure for future backend integration

---

## Learning Outcomes

Through this project, I improved my understanding of:

- Multi-page web architecture
- State management using LocalStorage
- Event handling & DOM manipulation
- Clean project structure
- Front-end debugging & UI improvement

---

## Future Vision

The long-term goal of this project is to transform it into a more complete full-stack e-commerce platform with authentication and backend integration.
